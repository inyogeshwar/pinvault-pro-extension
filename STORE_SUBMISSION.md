## ü¶ä Firefox Add-ons (AMO)

### Requirements
- Firefox Developer Account
- Extension signed by Mozilla
- Screenshots (up to 10)
- Detailed description

### Steps
1. **Create Developer Account**
   - Visit: https://addons.mozilla.org/developers/
   - Sign in with Firefox Account

2. **Submit Extension**
   - Click "Submit a New Add-on"
   - Upload your `.xpi` file
   - Choose distribution channel (AMO or Self-distribution)

3. **Fill Extension Details**
   - Name: "PinVault Pro"
   - Summary: "Professional Pinterest bulk image downloader"
   - Description: Use detailed feature list
   - Categories: "Download Management", "Photos & Media"
   - Tags: pinterest, downloader, bulk, images

4. **Upload Media**
   - Screenshots: Upload 3-5 screenshots showing the extension in action
   - Icon: Will use icon from manifest

5. **Review Process**
   - Automated review (minutes to hours)
   - Manual review if flagged (up to 14 days)
   - Address any reviewer feedback

## üåê Microsoft Edge Add-ons

### Requirements
- Microsoft Partner Center account
- Extension package (.zip)
- Screenshots and store assets
- Age rating and category selection

### Steps
1. **Create Partner Center Account**
   - Visit: https://partner.microsoft.com/dashboard/microsoftedge
   - Sign in with Microsoft account
   - Complete developer registration

2. **Submit Extension**
   - Click "Create new extension"
   - Upload your `PinVault-Pro-Edge-v1.2.0.zip`
   
3. **Store Listing**
   - Use same description as Opera
   - Category: "Developer tools" or "Productivity"
   - Age rating: "General audiences"

4. **Review Process**
   - Typically 7-14 days
   - Comprehensive security and policy review

## üü¢ Chrome Web Store

### Requirements
- Google Developer Account ($5 one-time fee)
- Extension package (.zip)
- Promotional images required
- Detailed store listing

### Steps
1. **Developer Registration**
   - Visit: https://chrome.google.com/webstore/devconsole
   - Pay $5 registration fee
   - Verify identity

2. **Upload Extension**
   - Click "Add new item"
   - Upload your Chrome-compatible package
   - Fill extensive store listing

3. **Review Process**
   - Can take 1-30 days
   - Very strict policy enforcement
   - May require multiple revisions
